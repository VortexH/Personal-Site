HBnB

Tech: Python, Flask, Javascript, jQuery, HTML, CSS

Deployment Tech: gunicorn, NGINX, Fabric, Bash scripts

Project Source Code: hbnb

This is a web application that I've developed from client to server side. Essentially, it displays rental unit listings from a database of entries containing units in various states and cities.

Features include:

  • console
  • JSON file storage
  • Database storage using SQLAlchemy
  • Amenity, City, State, Place, Review, User classes
  • Amenity filtration and dynamic loading of content using jQuery
  • a public API to view, create, and delete rental unit listings

Development of the project involved different team members for each phase of the project, unit testing, and pair programming.

Hosted project on a web architecture containing 1 load balancer and a 2 web servers. Used application server, gunicorn, for serving content through NGINX and Fabric to pack and unpack tarballs containing content.

Abalone

Tech: Python, Flask, Semantic UI, Javascript, jQuery, HTML, CSS, MySQL

Deployment Tech: gunicorn, NGINX

Project Source Code: abalone

Mission: Develop a web application allowing users to comment on youtube videos.

A group of 12 students from Holberton School developed the app as if we were all in a company. So that meant there were teams for: Front end, Back end, OAuth, Marketing, Project Management, and Site Reliability Engineering.

I was part of the backend developing the routes for posting and retrieving comments from a MySQL database. The routes would be accessible from the front end and act as the engine for the comment engine.

Learned the effects of a looming deadline on the project, development hiccups, among other realities of project development within a short timeline.
